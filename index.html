This game setting is set to a platform game and in a galaxy environment. The interface design of this game and the music in this game that match the environment can attract player to play in longer period. This game is supported the PC-Based Platform. Level design will give player a different game experience and more interesting game play and funny experience. Player just need to use space bar key to jump to avoid the obstacles and the down key is use to fall the character. During the game on going, player need to control the character to avoid the obstacles and collect as much coins as player can. Player win the game when character reach the end of the game level. There will be an UFO at the end of the game level.
